Para executar o projeto é necessário antes instalar as dependências pelo terminal com o seguinte comando:

npm install

O comando deve ser executado uma vez no diretório "/pfc-app/mobile/" e uma vez no diretório "/pfc-app/backend/".

Uma vez feito isso é preciso executar tanto o backend quanto o frontend com:

npm start

Obs: Precisa alterar o IP informado na pasta "src/services/api.js" no frontend
